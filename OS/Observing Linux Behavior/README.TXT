1. To run this program you must compile it useing this line in linux:
	"gcc observe.c -o observe"
2. To run the program you may do one of the following:
	1: "./observe" ~This will only print out part B.
	2: "./observe -s" ~This will print out part B and part C.
	3: "./observe -l -%d -%d" ~For the last command you may print part B, part C, and part D With the first digit must be the interval of updates for load averages and the second digit is the duration to print out the load averages.
